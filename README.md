### Tutorial Deploy di VPS


 • `git clone https://github.com/AyiinXd/DownloaderMedia`

 • `cd DownloaderMedia`

 • `pip3 install -U -r requirements.txt`

 • `cp sample.env .env`

 • `nano .env`
  - isi vars
  - Jika sudah 
  - ketik ctrl + S
  - ctrl + X

 • `screen -S DownloaderMedia`

 • `bash start` atau `python3 -m bot`
